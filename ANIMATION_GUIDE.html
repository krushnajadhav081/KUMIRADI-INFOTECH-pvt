Step 1: Add CSS File to Your HTML

Add this line in the <head> section of your HTML file:

Step 2: Add JavaScript File

Add this line before the closing </body> tag:

üêõ Troubleshooting

Animations not working?

  1. Check if CSS file is loaded: Open browser DevTools ‚Üí Network tab
  2. Check if JS file is loaded: Open Console, look for "Smooth Animations Loaded"
  3. Verify file paths are correct
  4. Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)

Animations too slow/fast?

Adjust the duration values in smooth-transitions.css

Want to disable specific animations?

Comment out or remove the specific animation class from your HTML
